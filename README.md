# GoEdit
This project needs a new name

### About
"GoEdit" is a terminal text editor written in Go.

### Install

	$ git clone https://github.com/hazbo/goedit.git
	$ cd goedit
	$ git submodule init
	$ git submodule update
	$ tools/configure
	$ make
	$ build/goedit
